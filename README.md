# Specific-style-transfer
This repository combines ideas from two papers to create a method for object-specific style transfer using Mask2Former for segmentation and VGG-19 for style transfer.

To run the code:

Follow the instructions directly in the code.

Upload the following files:

requirements.txt (provided)

A content image – any image you want to stylize

A style image – an image representing the style you want to apply

For multi-object style transfer:

Upload multiple style images if you want different styles for different objects.

If you want the same style applied to all objects without affecting the background, you only need a single style image.

Note: In the code, the uploaded images are referenced by their file extensions (e.g., .jpg or .png). Make sure to update these references to match the actual extensions of the images you upload.
in addition we uploaded couple of photos and styles for you to try.
